HTML

HTML stands for Hyper Text Markup Language,HTML is used to create webpages and for users to see on their phones, laptops, and Desktops. HTML pages contain HTML tags which usually have an opening part < > and a closing part < / >.

Elements

An HTML element is everything from the start tag to the end tag for example. < p > I am typing this to you < / p >
< p > <-- is the start tag and < / p > <-- is the end tag. Those two together make up one element.

Attributes

Elements in HTML have attributes; these are additional values that configure the elements or adjust their behavior in various ways to meet the criteria the users want.

The < p > tag defines a paragraph.


The h1 - h6 tags are used as headers for HTML pages. The lower the number the smaller the header.

Span

The span tag is used to style or edit inline elements. You can use it to change a very specific part of a paragraph without affecting other parts of the paragraph. I was able to change the color of the word 'affecting" without using changing the color of the other words in the paragraph because I used the span tag


Images

In HTML, images are defined with the < img > tag, which allows for the insertion of images by providing the images file location
