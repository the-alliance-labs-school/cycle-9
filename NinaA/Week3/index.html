HTML Reference Guide

HTML stands for "Hyper Text Markup Language".

HTML describes the structure of a Web page.

HTML Consist of a series of elements.

HTML elements tell the browser how to display content.

HTML elements are represented by tags. Opening< > and closing < / >

HTML attributes are within the element tags, specifically within the starting tag.

Headings

Your headings guide the reader's eyes to read what's next.

< h1 >Your text here< /h1 >

H1 Largest

H2 Large

H3 Medium

H4 Small

H5Smaller
H6 Smallest

Links

You can link to other portions of the document, page, or another website using the < a >< /a> tag.

To create a hyperlink, you use the tag in conjuction with the href attribute.

The href attribute's value is the URL or location of where the link is pointing to.

Should look like this: < a href="location"> Text you want displayed < /a >

back

linebreaks

Line breaks are necessary to seperate and not overcrowd a document

Using the tag < br > without a closing tag, places a break between two spaces.

back

Div

The < div > tag defines a division or a section in the HTML document.

The < div > element is often used as a container for other elelments to style them with css or to perform a certain tasks with Javascript.

back

Types of list

There are two types of list you can create in your html documents

There is ordered list & Unordered list.

< un >< /un> Would display a list with bulletpoints

< ol >< /ol> Would display a numbered list

Span

Span is an element is used to group inline-elements in a document.

The tag provides no change by itself, it adds a way to change a part of a text or of a document.

In the example below lets change the color of part of the text and not all.

My mother has blue eyes.

My mother has< span style="color:blue">blue< /span > eyes.

back

Media

Pictures & Videos are worth a million words.

We want to add media content to our webpages

We want to make unicorns appear our pages.

To do so we would use the < img > tag and only requires an opening tag.

Within the < img >tag you will include src= which is the source of the media alt=description width and height you want the media.

Formating text

Formating text in your page can brign attention to specific parts of your message.

There is Bold Italic Strike outHighlighted text Underlined, just to name a few.

< strong>< /strong> < b >< /b > < em>< /em> < i>< /i> < strike>< /strike> < mark>< /mark>

Using the tags strong or b for bold

Using the tags em or i. While both tags display the same result the difference is strong and em mean that the text is "important".

Using the tags strike will strike out the text within the element.

Using the tags mark will highlight your text.
